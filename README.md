# ✍️ 손해사정 블로그 리라이팅 어시스턴트 (무료)

다른 손해사정사 블로그의 최신 글을 가져와서, **내 "보상하다" 문체로 유사하지 않게 새로 써주는** 도구입니다.
Google Gemini의 **무료 API**를 사용하므로 비용이 들지 않습니다.

## 동작 방식

1. **STEP 1 — 문체 선택** — 4개 블로그 문체 중 하나를 고릅니다.
2. **STEP 2 — 블로그 주소 입력** — 참고할 다른 손해사정사 네이버 블로그 주소를 여러 개 넣습니다(한 줄에 하나).
3. **STEP 2 — 최신 글 5개 가져오기** — 각 블로그의 최신 손해사정 글 제목 5개를 자동으로 가져옵니다.
   (RSS 피드의 카테고리/제목으로 손해사정 글을 우선 추려냅니다.)
4. **STEP 3 — 제목 클릭** — 원하는 글 제목을 클릭하면, 그 글의 본문을 가져와
   **고른 문체의 인사말·말투·끝맺음**으로 **완전히 새로운 글**로 재작성합니다.
5. **복사 / .md 저장** — 결과를 복사하거나 파일로 저장합니다.

## 선택 가능한 4개 문체 (페르소나)

| 문체 | 블로그 | 특징 |
|------|--------|------|
| 안녕하다 (보상하다) | hello_hada | 친근+단호, 비유 많은 8단 공식 |
| 친절보상 | kind_bosang | 차분·정돈된 안내서 톤, 코드 대조 |
| 보상히어로 | kind_yk | 히어로 컨셉 자신감, 표·후유장해 챙김 |
| 법률사무소 마에스트로 | maestrolaw | 격식 법률체, '의뢰인' 호칭, 실제 사례+해시태그 |

각 문체의 인사말/끝맺음/말투는 `lib/personas.ts`에 정리돼 있습니다. 수정하거나 새 문체를 추가할 수 있습니다.

## 저장된 소스 블로그 (참고용 손해사정사 블로그 15곳)

STEP 2 화면에 손해사정사 블로그 15곳이 칩(chip)으로 저장돼 있습니다.
- 칩을 클릭하면 주소칸에 하나씩 추가됩니다.
- "전체 불러오기" 버튼으로 15곳을 한 번에 넣습니다(0.4초 내 동시 처리).
- 목록을 바꾸려면 `app/page.tsx`의 `SOURCE_BLOGS` 배열을 수정하세요.

## 준비물

- [Node.js](https://nodejs.org/) (이미 설치되어 있음)
- Google Gemini API 키 → https://aistudio.google.com/app/apikey 에서 **무료** 발급

## 처음 설정 (한 번만)

1. `.env.local` 파일을 열고 발급받은 키를 넣으세요.
   ```
   GEMINI_API_KEY=AIza... (발급받은 키)
   ```

## 실행 방법

- **가장 쉬운 방법:** `run.bat` 파일을 더블클릭하세요.
- 또는 터미널에서: `npm install` (처음 한 번) → `npm run dev`

실행되면 브라우저에서 **http://localhost:3000** 을 엽니다.

## 문체는 어떻게 재현하나요?

- 각 블로그의 실제 글을 분석해 인사말·끝맺음·문체 가이드·견본을 `lib/personas.ts`에 정리해 두었습니다.
- "안녕하다(보상하다)" 문체는 추가로 `data/persona.json`(과거 글 846편)에서 손해사정 글 2편을 무작위로 뽑아 견본으로 사용합니다.
- 문체를 바꾸거나 새 블로그를 추가하려면 `lib/personas.ts`의 `PERSONAS` 배열을 수정하세요.

## 폴더 구조

```
app/
  page.tsx                   메인 화면 (문체 선택 → 주소 → 목록 → 리라이팅)
  api/fetch-titles/route.ts  블로그 최신 글 목록 가져오기 (RSS)
  api/rewrite/route.ts       원문 가져와 선택한 문체로 재작성 (Gemini)
  api/generate/route.ts      (옵션) 주제만으로 글 생성
lib/
  naver.ts                   네이버 블로그 스크래핑 (RSS + PostView)
  personas.ts                4개 블로그 문체(인사말/끝맺음/가이드/견본)
  persona.ts                 안녕하다 문체용 persona.json 견본 추출
data/
  persona.json               안녕하다(보상하다) 과거 글 846편
```

## 비용 / 무료 한도

Gemini 무료 등급은 요금이 청구되지 않습니다. 다만 분당/하루 요청 수 한도가 있어,
글을 연달아 많이 뽑으면 잠시 멈췄다가 다시 시도해야 할 수 있습니다.

## 주의

- 네이버 블로그가 **비공개**이거나 **이미지로만** 된 글은 본문을 가져올 수 없습니다.
- 생성된 글은 사실(질병코드·약관·판례 등)이 정확한지 게시 전 직접 한 번 검토하세요.
